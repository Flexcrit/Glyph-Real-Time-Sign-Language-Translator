# Glyph-Real-Time-ASL-to-Text-Translator
Glyph is a high-performance computer vision application that translates American Sign Language (ASL) into text instantly. It identifies hand gestures for the alphabet (A-Z) with high precision to make digital and physical spaces accessible for everyone.

Features
Instant Translation: Converts hand signs to English letters in real time with minimal latency.

A-Z Coverage: Specifically recognizes and translates individual letters from the American Sign Language alphabet.

Computer Vision Power: Uses advanced image processing to track hand movements accurately.

Seamless Accessibility: Enables hearing-impaired users to spell out words and sentences effortlessly.

Technical Stack
Machine Learning: TensorFlow and Mediapipe.

Frontend: Next.js and TypeScript.

Backend: Supabase and PostgreSQL.

Computer Vision: OpenCV.

Impact
Glyph empowers independence in healthcare, education, and daily retail interactions. It reduces the need for human interpreters by providing a portable, digital bridge. This project transforms how the world interacts by turning gestures into a universal language.

Installation
Clone the repository to your local machine.

Install the necessary dependencies using your package manager.

Run the application to start the live camera feed for gesture recognition.
